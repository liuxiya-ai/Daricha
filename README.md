## About This Project
Daricha is a **minimalist, offline-ready educational navigation hub** designed for users in low-bandwidth environments (specifically Afghanistan).

It aggregates:
*   Digital Schooling Resources (K-12 & University)
*   Digital Safety Guides
*   Mental Health Support
*   Language Learning Tools

## Technical Philosophy
*   **Zero-JS Core**: The site functions perfectly without JavaScript (except for the panic button).
*   **<10KB Load**: Extremely lightweight HTML/CSS.
*   **Privacy First**: No analytics, no cookies, no tracking.

## Local Development
To run this site locally:
1.  Clone the repository.
2.  Open `index.html` in any browser.
3.  That's it. No build step required.
